
# ZTB Serial Protocol

ZTB means Zephyr Test Bench.

The protocol is a simple text-based command protocol over UART.
Each command is one line terminated by newline.

## Frame Format

```text
ZTB|seq=<number>|cmd=<command>|key=value|key=value
````

## Response Format

```text
ZTB|seq=<number>|status=<OK|FAIL>|key=value|key=value
```

## Common Fields

| Field  | Description                                  |
| ------ | -------------------------------------------- |
| ZTB    | Protocol prefix                              |
| seq    | Sequence number generated by Robot Framework |
| cmd    | Command name                                 |
| status | Response status: OK or FAIL                  |
| err    | Error code if status is FAIL                 |

---

# GPIO Commands

## GPIO_WRITE

Set a digital output channel.

Command:

```text
ZTB|seq=1|cmd=GPIO_WRITE|ch=DIO_OUT1|val=1
```

Response:

```text
ZTB|seq=1|status=OK
```

## GPIO_READ

Read a digital input channel.

Command:

```text
ZTB|seq=2|cmd=GPIO_READ|ch=DIO_IN1
```

Response:

```text
ZTB|seq=2|status=OK|val=1
```

## GPIO_EXPECT

Wait until a digital input channel matches the expected value or timeout expires.

Command:

```text
ZTB|seq=3|cmd=GPIO_EXPECT|ch=DIO_IN1|val=1|timeout=100
```

Pass response:

```text
ZTB|seq=3|status=OK|expected=1|actual=1
```

Fail response:

```text
ZTB|seq=3|status=FAIL|expected=1|actual=0|err=GPIO_MISMATCH
```

# Testbench Resource Map

Robot Framework uses logical resource names instead of physical pin numbers.

| Resource  | ESP32 Pin | Direction | Purpose                           |
| --------- | --------- | --------- | --------------------------------- |
| DIO_OUT1  | GPIO27    | Output    | Simulated digital input to DUT    |
| DIO_IN1   | GPIO14    | Input     | Digital output monitored from DUT |
| DAC_OUT1  | GPIO25    | Output    | Analog stimulus to DUT ADC        |
| DIO_IN2   | GPIO34    | Input     | Warning/status monitor from DUT   |
| UART1_TX  | GPIO17    | Output    | UART TX to DUT                    |
| UART1_RX  | GPIO16    | Input     | UART RX from DUT                  |
| SPI1_SCK  | GPIO18    | Output    | SPI clock                         |
| SPI1_MISO | GPIO19    | Input     | SPI MISO                          |
| SPI1_MOSI | GPIO23    | Output    | SPI MOSI                          |
| SPI1_CS   | GPIO21    | Output    | SPI chip select                   |

# Example DIO Test Sequence

## TC_DIO_001_BUTTON_HIGH_LED_ON

```text
ZTB|seq=1|cmd=GPIO_WRITE|ch=DIO_OUT1|val=1
ZTB|seq=2|cmd=GPIO_EXPECT|ch=DIO_IN1|val=1|timeout=100
```

## TC_DIO_002_BUTTON_LOW_LED_OFF

```text
ZTB|seq=3|cmd=GPIO_WRITE|ch=DIO_OUT1|val=0
ZTB|seq=4|cmd=GPIO_EXPECT|ch=DIO_IN1|val=0|timeout=100
```
